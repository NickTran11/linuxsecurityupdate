Quick install

# Option A: Download from this site
curl -L -o linuxsecurityupdate.py "$(python3 -c 'print("https://nicktran11.github.io/linuxsecurityupdate/downloads/linuxsecurityupdate.py")')"

# Run
sudo python3 linuxsecurityupdate.py --sudo

Replace YOUR_PAGES_URL with your GitHub Pages URL.

What you get

  • Creates a Linux user and sets a password
  • Installs & enables OpenSSH server
  • Backs up and updates /etc/ssh/sshd_config
  • Disables root SSH login (PermitRootLogin no)
  • Optional UFW rule for OpenSSH

Safety reminders

This script modifies system configuration and requires root. Review the code before running. Don‚Äôt hardcode real credentials in production.
